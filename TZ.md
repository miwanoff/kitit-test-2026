# ТЗ: Оновлення KitIT Test v2 (Лідогенерація)

## 🎯 Мета проєкту
Трансформувати процес завершення тесту: замінити пряме завантаження PDF-сертифіката на форму збору даних (лідогенерацію) з подальшою видачею персоналізованого посилання на Telegram-бот.

## 🛠 Технологічний стек
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla), Bootstrap 5.
- **Backend/Storage:** Google Apps Script (Web App), Google Sheets.
- **Інтеграція:** Deep-linking для Telegram.
- **Інструменти розробки:** VS Code + Roo Code / Gemini 2.5 Flash.

## 📋 Функціональні вимоги

### 1. Екран результатів (оновлення)
Після проходження тесту користувач бачить свій архетип та рекомендації (як і раніше), але кнопка завантаження сертифіката замінюється на **Форму реєстрації**.

### 2. Форма збору лідів
Додати блок форми після результатів тесту з наступними полями:
- `Ім'я` (text, required)
- `Прізвище` (text, required)
- `Телефон / Telegram` (text, required)
- `Вік` (number, required)

### 3. Обробка даних (Google Sheets API)
- При сабміті форми дані повинні надсилатися на Google Apps Script Web App URL через `fetch` (POST).
- **Дані для відправки:**
    - Дата (генерується на стороні скрипта або JS).
    - Ім'я, Прізвище, Контакт, Вік.
    - Результат тесту (Архетип).
    - Рекомендований напрямок та курс.
- Скрипт має повернути ID рядка або статус успіху.

### 4. Фінальний CTA (Call to Action)
Після успішної відправки форми:
1. Форма зникає.
2. З'являється велика кнопка **"Отримати сертифікат у Telegram"**.
3. **Deep-link логіка:** Посилання має формат `https://t.me/KIT_IT_BOT?start={archetype}_{rowID}`, де:
   - `{archetype}` — код архетипу (a, b, c, d, e).
   - `{rowID}` — унікальний ідентифікатор запису (номер рядка з таблиці).

## 🎨 Дизайн та UX
- Використовувати існуючу кольорову гаму (основний блакитний `#87D7FC`).
- Форма повинна бути адаптивною (mobile-first).
- Кнопка Telegram має містити відповідну іконку та виділятися візуально.
- Додати індикатор завантаження (spinner) під час відправки даних у таблицю.

## 🗂 Структура бази даних (Google Sheets)
Таблиця повинна мати наступні заголовки:
`Дата | Ім'я | Прізвище | Контакт (Тел/ТГ) | Вік | Архетип | Напрямок | Курс`
